# 《我们之间》TTS 音色 Prompt 设计

> 共 5 个音色（林远分少年/成年两版），适配阿里云 TTS 语音模型

## 一、旁白（叙述者）

### 音色 Prompt

```
角色身份：第三人称全知叙述者，故事的讲述人，类似纪录片旁白或文学有声书朗读者。

年龄性别：男性，约45-55岁，声音成熟厚重。

音色特征：中低音域，嗓音浑厚、沉稳、有磁性，带有岁月沉淀的质感。声线干净，无明显沙哑或气声，共鸣集中在胸腔，给人可靠、值得信赖的听感。

口音语言：标准普通话，不带明显地域口音。咬字清晰但不刻意，自然流畅，像在讲述一个自己亲眼见证过的故事。

语气情绪：整体基调克制内敛，不煽情、不夸张。叙述平静段落时语气淡然从容，像老友闲聊；涉及情感转折时（如"塔是送电的，不是送话的"）略微放慢，声音微微压低，带一丝不易察觉的温度。全程保持"旁观者"的距离感，不代入任何角色立场。

语速音量：语速偏慢，约每分钟160-180字。句与句之间有自然停顿，段落转换时停顿稍长（0.8-1.2秒）。音量中等偏轻，不抢戏，适合作为背景叙述层。

适配场景：全篇贯穿，覆盖所有6个章节。适合在安静的室内环境、夜晚氛围中聆听。与像素风视觉小说的画面节奏配合，文字逐字显示时语音同步推进。
```

### 关键情绪节点

| 场景ID | 情绪标记 | 处理建议 |
|--------|----------|----------|
| 0-4 | 平静开场 | 正常语速，轻松自然 |
| 12 | 微微感慨 | "那是第一次"稍慢，轻微叹息感 |
| 22 | 带一点好奇 | 模拟孩子提问的语气，但不变成孩子声音 |
| 36 | 温暖安静 | 放慢，声音略柔 |
| 51-52 | 全篇情感核心 | "塔是送电的，不是送话的"——最慢语速，每个字之间有微停顿，声音低沉但清晰 |
| 68-69 | 回忆涌上 | 语速渐慢，声音带一丝颤抖的温度 |
| 78 | 收束余韵 | 最后一句旁白，语速最慢，像画面定格 |

## 二、老周（线路工）

### 音色 Prompt

```
角色身份：基层电力线路巡检工，在旧仓村巡线二十余年。沉默寡言，不善表达情感，但内心细腻。是一个典型的中国农村中年技术工人形象。

年龄性别：男性，约50-55岁。

音色特征：中低音域，嗓音偏粗、偏哑，有长期户外工作留下的风霜感。声音结实但不洪亮，像被风吹过的旧木头。胸腔共鸣为主，偶尔带一点喉部的颗粒感。不圆润，不修饰，质朴真实。

口音语言：普通话为主，但带有轻微的南方农村口音痕迹（如平翘舌不完全区分、"的"偶尔弱读）。用词极简，从不说废话，句子短促。说话像在干活——干脆利落，不拖泥带水。

语气情绪：极度克制，几乎不带情绪波动。回答问题时像在陈述事实，不解释、不安慰、不感叹。但克制之下有温度——比如"好"（场景43）虽然只有一个字，语气里有一丝答应的暖意；"现在呢"（场景71）虽然只有三个字，声音微微上扬，带一点期待和欣慰。

语速音量：语速偏慢，约每分钟140-160字。说话时字与字之间间隔均匀，像在一步一步走路。音量中等偏低，不刻意提高声量，即使简短的回答也不会突然变大。

适配场景：与林远的对话场景（山坡、塔基、村口）。声音需要和旁白形成区分——旁白是"讲述"，老周是"说话"，老周的声音应该更接地气、更粗糙、更短促。
```

## 三、林远·少年（12-14岁）

### 音色 Prompt

```
角色身份：农村留守儿童，12-14岁，跟外婆住在旧仓村。父母在外打工，暑假帮家里干活。对世界充满好奇，但性格偏安静内向，不聒噪。

年龄性别：男性，12-14岁少年。

音色特征：中高音域，少年清亮的嗓音，但不过分尖锐或稚嫩。声音干净透明，带一点未变声期尾段的青涩感。不奶声奶气，像一个已经懂事的孩子——声音里有超出年龄的安静。

口音语言：普通话，带有轻微的农村孩子口语习惯（如"叔"的称呼自然亲切）。问句多，语气里带着真诚的好奇，不是撒娇式的提问，而是认真想知道答案的那种问法。

语气情绪：整体基调是好奇+孤独。对老周、对输电塔、对外面的世界充满好奇，问问题时眼睛是亮的；但底色里有一层淡淡的孤独——父母不在身边，暑假在花生地里发呆的那种孤独。"那它一晚上闪多少下？"这句话里，既有孩子天真的好奇，也有一个留守孩子试图理解远方父亲的小心思。

语速音量：语速正常偏快，约每分钟180-200字。孩子说话比大人快，但林远不是话多的孩子，所以每句话之间有自然的间隔。音量中等，不喊叫，像在和信任的人说话。

适配场景：山坡初遇、塔基对话、村口偶遇。声音需要和成年林远形成对比——少年更清亮、更快、更外露；成年更沉稳、更慢、更内敛。
```

### 关键情绪节点

| 场景ID | 台词 | 情绪标记 | 处理建议 |
|--------|------|----------|----------|
| 5 | 叔，你是供电局的？ | 好奇搭话 | 语气轻快，像跑过来喘着气问的 |
| 7 | 塔上那个，那个红色的灯，晚上会闪的，是干什么用的？ | 认真好奇 | "那个"重复一次，像在想怎么描述 |
| 9 | 哦。那它一晚上闪多少下？ | 天真追问 | "哦"是恍然大悟，然后紧接着追问，语速稍快 |
| 28 | 我爸寄回来的。 | 平静陈述 | 语气平淡，像在说一件习惯了的事 |
| 30 | 深圳。他说今年又不回来过年。 | 失落 | "深圳"两个字后有一个小停顿，"又不"微微加重 |
| 37 | 那它闪灯的时候，我爸能不能看见？ | 全篇最触动的一句 | 声音微微发颤，不是哭，是期待和害怕同时存在。语速放慢 |
| 42 | 那你后天来的时候，能不能跟我说说，这塔是怎么架起来的。 | 期待 | 语气比前面轻快，像找到了一个可以期待的事情 |
| 50 | 说完了，就挂。挂了更没意思。 | 少年老成 | 语气平淡，像已经接受了这种状态 |

## 四、林远·成年（22岁）

### 音色 Prompt

```
角色身份：大学毕业的电气工程专业学生，回到旧仓村跟随县供电局实习。六年后重回故地，从好奇的留守少年成长为有专业素养的青年工程师。

年龄性别：男性，22岁。

音色特征：中音域，青年男性的声音，清朗但不单薄。比少年时期低沉了一个台阶，有了胸腔共鸣的厚度，但依然保持着干净的底色。声音里有书卷气，也有从父亲和老周那里继承的朴实感。

口音语言：标准普通话，受过高等教育后的语言习惯。用词比少年时期更完整、更有条理，但面对老周时依然自然地叫"周叔"，保留着乡村孩子的亲切感。

语气情绪：整体基调是沉稳+感慨。六年的成长让他不再像孩子一样追问，而是学会了表达和倾听。"现在，我觉得它不是把人送走的。是把人叫回来的。"——这句话是全篇的情感高潮，语气应该是平静的、确定的、温暖的，像一个人终于想明白了一件事。

语速音量：语速正常，约每分钟170-190字。比少年时期慢了半拍，说话更有条理，句与句之间有思考的停顿。音量中等，不刻意压低也不刻意放大。

适配场景：2025年冬天的归来章节。与老周在塔下重逢的对话。声音需要同时承载"重逢的温暖"和"成长的沉淀"。
```

| 场景ID | 台词 | 情绪标记 | 处理建议 |
|--------|------|----------|----------|
| 64 | 周叔。 | 重逢 | 两个字，声音微微上扬，带笑意，像远远看到就喊了一声 |
| 66 | 我大学毕业了，学的电气工程。这次回来，是跟着县供电局实习。 | 自豪但克制 | 语速正常，像在汇报，但不炫耀 |
| 67 | 我想看看这些塔。 | 真诚 | 简单直接，声音里有感情但不外露 |
| 70 | 周叔，我以前觉得，这些塔把我家的地占了，我爸才走的。 | 回忆与坦诚 | 语速稍慢，像在回忆往事，声音平静 |
| 72 | 现在，我觉得它不是把人送走的。是把人叫回来的。 | 全篇高潮 | "现在"后微停顿，"不是把人送走的"语速正常，"是把人叫回来的"放慢、加重、温暖。声音稳定，不颤抖，是确定的 |
| 75 | 画的。 | 简洁回应 | 两个字，语气轻快自然，像笑着回答 |

## 五、老板娘（小卖部）

### 音色 Prompt

```
角色身份：旧仓村村口小卖部的中年女性经营者。热情健谈，消息灵通，是村里的"信息中转站"。典型的农村热心大姐形象。

年龄性别：女性，约40-50岁。

音色特征：中高音域，女性声音明亮爽朗，有烟火气。嗓音不尖不细，带一点中年的圆润感。笑起来声音会往上扬，有感染力。整体听感亲切、接地气，像邻家大姐在跟你聊天。

口音语言：普通话为主，带有明显的地方口语色彩。语气词多（"呀""呢"），说话像在唠家常，不书面化。句子短，节奏快，像连珠炮一样把消息倒出来。

语气情绪：热情、日常、没有心机。说"就是你们那个输电塔呀"的时候带着一点骄傲（"我们村有这个东西"），说"林远他爸回来了"的时候带着替人高兴的语气。整体情绪偏积极正面，是故事里最轻松的声音。

语速音量：语速偏快，约每分钟200-220字。说话节奏明快，句与句之间衔接紧密，像怕冷场一样。音量中等偏大，比所有其他角色都略响一点，符合"开小卖部的人说话中气足"的特征。

适配场景：村口小卖部，2021年变化章节。只出现3句台词，但需要在一众低沉男声中跳出来，给听众一个"换了口气"的感觉。
```

| 场景ID | 台词 | 情绪标记 | 处理建议 |

| 57 | 就是你们那个输电塔呀。有些人觉得好看，拍照。 | 自豪闲聊 | 语气轻快，"呀"字上扬，像在炫耀村里的景点 |
| 59 | 林远他爸回来了。在村里包了块地养鱼，还弄了个农家乐。 | 替人高兴 | 语速稍快，像在传递好消息，声音里有笑意 |
| 61 | 读书呀，县里。放假才回来。 | 日常回答 | 语气自然，像在回答邻居的随口一问 |

> **注意**：以上 voice 名称为参考建议，实际需根据阿里云 TTS 控制台当前可用的音色列表进行选择。建议先在控制台试听后，选择最接近上述描述的音色，再通过 speed/volume/pitch 参数微调。

### 音频文件命名规范

```
tts_audio/
├── narrator/          # 旁白
│   ├── scene_000.mp3
│   ├── scene_001.mp3
│   └── ...
├── oldzhou/           # 老周
│   ├── scene_008.mp3
│   ├── scene_011.mp3
│   └── ...
├── linyuan_young/     # 林远·少年
│   ├── scene_005.mp3
│   ├── scene_007.mp3
│   └── ...
├── linyuan_adult/     # 林远·成年
│   ├── scene_064.mp3
│   ├── scene_066.mp3
│   └── ...
└── shopkeeper/        # 老板娘
    ├── scene_057.mp3

```

文件名中的数字对应 `script.js` 中的 `scene.id`，方便后续在代码中通过 `scene_${id}` 直接匹配音频文件。

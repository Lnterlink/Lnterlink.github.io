# 《我们之间》MIMO TTS 音色设计 Prompt

> 工作流：音色设计 → 音色复刻（两步法保证一致性）
> 工具版本：v2.0（双Tab工作流）

## 工作流说明

### 为什么需要两步？

音色设计模式（voicedesign）每次生成都是"重新设计"声音，即使描述相同，不同台词的音色也会变化。
解决方案：**先用设计模式创建一条满意的参考音频，再用复刻模式（voiceclone）基于该音频生成全部台词**。

### Step 1：音色设计（创建参考音频）

1. 打开工具，进入 **「Step 1 · 音色设计」** 标签页
2. 选择角色 → 点击「填入预设描述」（或手动粘贴下方音色描述）
3. 输入一段试听文本（建议选该角色有代表性的台词）
4. 点击「🔊 生成试听」→ 试听效果
5. 不满意就调整描述后重新生成，反复迭代
6. 满意后点击 **「💾 存为参考音频」** — 自动保存到 `tts_output/_references/`

### Step 2：批量生成（音色复刻）

1. 切换到 **「Step 2 · 批量生成」** 标签页
2. 选择角色 → 参考音频会自动加载（Step 1 保存的）
3. 点击「📝 从台词文档生成」→ 自动生成各角色的批次文件
4. 在左侧选中批次文件 → 点击「📁 加载」→ 台词列表显示在右侧
5. 点击「全选」→ 点击 **「🔊 批量生成选中」**
6. 音频自动保存到 `tts_output/{角色目录}/scene_XXX.wav`

### 技术说明

- **音色描述**：通过 `user` 消息传递，建议 1-4 句话，突出关键特征
- **风格指令**：设计模式下可逐句调整；批量复刻模式下暂不使用
- `voicedesign` 模式的 `audio` 对象中不需要 `voice` 参数
- 避免矛盾特征、后期术语（混响/回声/EQ）、模糊词汇
